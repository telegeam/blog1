你来到这个页面，通常有两个原因。

## 一、链接错误

原因：博客搬迁造成的旧链接失效。

例子：《中文文案排版指北（简体中文版）》的链接是

```
http://mzlog.com/wiki/chinese-copywriting-guidelines/
```

请修改成

```
https://telegeam.github.io/wiki/chinese-copywriting-guidelines/
```

改动要点：将域名 `mzlog.com` 改成 `telegeam.github.io`。

**备用办法：** 在「[分类](/categories/)」中找。

## 二、维度攻击

不明力量入侵，你想要访问的页面被摧毁了。
