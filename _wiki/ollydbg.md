### 快捷键

Ctrl --> C

Shift --> S

Alt --> M

| 功能               | 快捷键 |
|:-------------------|:-------|
| 设置/取消断点      | F2     |
| 执行到光标所在行   | F4     |
| 步过               | F8     |
| 步进               | F7     |
| 运行               | F9     |
| 暂停               | F12    |
| 回到应用层         | M-F9   |
| 打开文件           | F3     |
| 重新调试           | C-F2   |
| 打开应用程序输入表 | C-n    |
| 寻找表达式         | C-g    |
| 打开断点窗口       | M-b    |
| 切换断点状态       | 空格   |
| 添加备注           | ;      |
